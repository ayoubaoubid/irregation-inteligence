The Agricultural Dataset

Understanding the foundation of our predictive machine learning model.

1. Dataset Introduction

The "AgriSmart-Climate" dataset is a comprehensive collection of time-series observations gathered from multiple IoT sensor nodes distributed across hypothetical experimental farm plots. It spans 24 months of continuous hourly measurements, providing the robust statistical foundation required to train highly accurate supervised learning algorithms.

2. Primary Data Sources

  • Soil Sensors: Capacitive moisture sensors planted at depth (20cm & 60cm).
  • Weather Stations: On-site microclimate telemetry for Temp, Humidity, and Wind speed.
  • Rain Gauges: Specialized tipping-bucket sensors measuring pluvial precipitation.

6. Data Preprocessing

Prior to machine learning, raw telemetry data undergoes strict preprocessing:

  • Missing value interpolation via K-Nearest Neighbors (KNN).
  • Outlier removal using Isolation Forest.
  • StandardScaler applied to continuous numeric features.

3. Feature Descriptions

Feature Name Unit / Type Description
temperature Celsius (°C) Ambient air temperature at 2m above ground.
humidity Percentage (%) Relative atmospheric humidity.
soil_moisture Percentage (% v/v) Volumetric water content in the root zone.
rainfall Millimeters (mm) Accumulated precipitation over past 24h.
wind_speed km/h Affects evapotranspiration rates.
needs_irrigation (TARGET) Binary (0/1) Human expert classification for required watering.

7. Feature Engineering

To boost model performance, we derive new features from the raw data. Evapotranspiration Index (ET0) is synthetically calculated from temperature, humidity, and wind speed. Rolling Averages (3-day and 7-day) of soil moisture are computed to give the model historical trend context, preventing reactive over-irrigation from single anomalous readings.

8. Why High-Quality Data Matters

In machine learning, "garbage in equals garbage out." A highly robust, well-distributed dataset covering all four seasons prevents the model from overfitting to summer droughts or spring floods. This ensures the AgriSmart System remains reliable year-round.
